# 劳权智助 · 功能验证材料包

本目录用于功能演示、回归测试、阶段验收。建议按编号顺序执行。

## 🎯 使用方式（推荐）

1. 先完成 [00-验证环境与账号](./00-验证环境与账号.md)。
2. 按 `01 → 10` 顺序逐项执行，每条用例都填写「结果」列。
3. 对失败项补充截图编号或报错信息（建议写在“备注”列）。
4. 附录文本可直接复制用于页面输入，减少临时造数据成本。
