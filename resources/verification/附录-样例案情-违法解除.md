# 附录 · 样例案情：违法解除劳动合同

> **用途**：维权前台、分析案情、生成文书  
> **人物与金额均为虚构**

---

## 一、案情摘要（短版）

2023年7月我入职某电商公司任运营专员，签订了三年劳动合同，月薪8000元。2026年4月15日公司以"业绩不达标"为由口头通知我当天离职，未出具书面解除通知，也未支付经济补偿。我认为属于违法解除，想主张赔偿金。

---

## 二、案情详述

### 基本情况

| 项目 | 内容 |
|------|------|
| 劳动者 | 张某某，男，1995年生 |
| 用人单位 | 某电商科技有限公司 |
| 入职时间 | 2023年7月1日 |
| 合同期限 | 2023年7月1日至2026年6月30日（三年） |
| 岗位 | 运营专员 |
| 月薪 | 8000 元（税前，银行转账） |
| 社保 | 正常缴纳 |

### 解除经过

2026年4月15日上午，部门经理口头通知张某某："最近业绩不达标，今天办理离职，不用再来上班了。"  

- 未出具 **书面解除劳动合同通知**  
- 未说明具体业绩指标及考核依据  
- 未办理离职手续、未结清工资  
- 未支付 **经济补偿金** 或 **赔偿金**

张某某认为公司构成 **违法解除劳动合同**。

### 法律争点（供分析案情 / 答辩）

1. 以"业绩不达标"解除是否满足法定要件？  
2. 口头通知是否发生解除效力？  
3. 劳动者可主张 **赔偿金**（2N）还是 **经济补偿**（N）？  
4. 如何计算：工作年限 2年10个月，月薪 8000 元；  
5. 仲裁时效：知道或应当知道权利被侵害之日起 **1 年**。

### 参考计算（演示用）

| 项目 | 计算 |
|------|------|
| 工作年限 | 2年10个月 → 按 3 年计算（N=3） |
| 经济补偿 | 8000 × 3 = 24000 元 |
| 违法解除赔偿金 | 8000 × 3 × 2 = **48000 元** |

> 实际以仲裁委认定为准，此处仅供 AI 输出核对。

### 建议补充证据

- 劳动合同原件  
- 工资流水、社保记录  
- 绩效考核制度及本人考核结果  
- 解除当日的录音、聊天记录  
- 同事证言（了解解除过程）
