**案由**：入职后未签书面劳动合同，主张二倍工资差额

---

## 一、走哪个入口？

| 项目 | 内容 |
|------|------|
| **推荐** | 首页 **开始维权** → **普通入口** |

## 二、普通入口怎么填？

在案情描述框中 **整段粘贴**（无需分字段）：

```

```

## 四、建议验证点

1. AI 识别 **未签劳动合同二倍工资** 请求。  
2. 提示 **劳动关系起止时间**、**工资基数** 举证要点。  
3. 推荐 **仲裁申请书** 及赔偿计算。  
4. 提示 **一年仲裁时效** 与证据补强建议。
