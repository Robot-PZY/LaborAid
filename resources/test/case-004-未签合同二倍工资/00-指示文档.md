# Case 004 · 未签合同二倍工资（餐饮门店）

## 1) 给前台的提示词（描述案件）

我叫何洋，2025年11月起在“锦味餐饮管理（苏州）有限公司”门店做后厨，入职后一直没有签劳动合同。  
工资是老板通过微信和银行卡转账混合发放，月薪大概6500元。  
我在2026年4月提出补签合同，公司只口头答应，后来一直拖。  
现在我准备离职，想主张未签劳动合同期间的二倍工资差额。  
我手里有入职聊天记录、工资转账截图和门店排班照片，但没有正式工牌与考勤系统导出。
