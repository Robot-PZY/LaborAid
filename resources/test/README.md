# 测试案件材料集

本目录用于前台演示、答辩展示与端到端联调。每个案件**一个文件夹、一层文件**，不再嵌套 `materials/` 子目录。

## 目录规范

```
case-xxx-案件简称/
├── 00-测试指引.md      ← 必读：走哪个入口、怎么填表、上传哪些证据
├── 01-案件摘要.md
├── 02-当事人信息.md
├── 03-…（文字证据）
├── …
└── *.png / *.svg       ← 图片类证据（与文字材料配套）
```

## 使用方式

1. 打开对应案件的 **`00-测试指引.md`**，按「系统入口 → 填表 → 建案 → 上传证据」顺序操作。
2. 文字证据可直接上传 `.md`，或自行导出 PDF 后上传。
3. 图片证据优先上传同名的 `.png`；若无 PNG 则上传 `.svg`。
