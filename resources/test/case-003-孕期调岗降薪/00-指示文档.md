# Case 003 · 孕期调岗降薪（电商客服主管）

## 1) 给前台的提示词（描述案件）

我叫陈雯，2024年入职“禾木电子商务有限公司”，岗位是客服主管，税前月薪15000元。  
2026年3月我告知公司已怀孕，4月开始公司安排我从主管岗位调到“客服支持岗”，并将月薪调整到9000元，理由是“业务结构调整”。  
调岗前没有和我协商一致，也没有提供岗位价值评估说明。  
我目前保留了调岗通知截图、近6个月工资对比明细、产检请假审批记录。  
我想确认：公司这样调岗降薪是否违法，我应当如何主张恢复原待遇或要求补差。
