# Case 001 · 工地欠薪（农民工）

## 1) 给前台的提示词（描述案件）

我是李强，2026年1月起在“华城建设劳务有限公司”承包的工地做钢筋工，和现场包工头赵建国口头约定按月结工资，月均约8000元。  
到2026年4月底，1-3月工资一直没结清，累计约24000元。我还在岗，多次微信催要，对方一直回复“工程款没下来，先等等”。  
我手里有聊天记录截图、一份工资发放明细图片、银行卡流水摘要。没有正式劳动合同，也没有完整考勤表。  
我想先判断：是先走劳动监察投诉，还是直接申请仲裁；如果要仲裁，申请书和证据清单怎么准备更稳妥。
