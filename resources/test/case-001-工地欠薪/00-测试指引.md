# Case 001 · 工地欠薪 — 测试指引

**当事人**：李强 · 钢筋工 · 华城建设劳务有限公司（包工头赵建国）  
**案由**：拖欠劳动报酬（约 2.4 万元，2026 年 1–3 月）

---

## 一、走哪个入口？

| 项目 | 内容 |
|------|------|
| **推荐** | 首页 **开始维权** → **专项维权** |
| **身份专区** | **农民工维权专区** |
| **具体情形** | **包工头 / 小老板欠薪** |
| **备选** | 普通入口（见文末「案情描述」整段粘贴） |

> 路径：首页 `/` → 点「专项维权」→ 选「农民工维权专区」→ 选「包工头 / 小老板欠薪」→ 填写下方表单 → 生成维权计划 → **创建案件**。

### 补充说明全文（可复制）

```
我是李强，2026年1月起在华城新区二期土建项目做钢筋工，与现场包工头赵建国口头约定月薪约8000元。2026年1–3月工资均未结清，累计约24000元。我仍在岗，多次微信催要，对方称工程款未下来。手中有微信催薪记录、工资发放明细图片和银行流水摘要，无书面劳动合同和完整考勤表。想确认先走劳动监察还是直接仲裁，以及如何准备申请书和证据清单。
```

## 三、建案后上传哪些证据？

进入 **我的记录** → 打开本案件 → **整理证据**（`/evidence`），按下列顺序添加并上传对应文件。

| 序号 | 证据名称（上传时填写） | 类型 | 文件 | 必传 | 说明 |
|------|------------------------|------|------|:----:|------|
| 1 | 微信催薪聊天记录 | 电子数据 | `03-微信催薪记录.md` 或 `微信聊天截图.png` | ✅ | 证明催要事实与欠薪态度 |
| 2 | 工资发放明细 | 书证 | `工资发放明细.png` / `工资发放明细.svg` | ✅ | 证明应发、未发金额 |
| 3 | 银行流水摘要 | 书证 | `04-银行流水摘要.md` | 建议 | 辅助证明既往发放习惯 |
| 4 | 案件背景材料 | 书证 | `01-案件摘要.md`、`02-当事人信息.md` | 可选 | 便于 AI 分析对齐案情 |

### 故意不上传（测「缺失提示」）

- 书面劳动合同  
- 完整考勤打卡记录  
- 工地入场登记表 / 工牌  

---

## 四、建议验证点

1. 案由识别为 **拖欠劳动报酬 / 欠薪**。  
2. 材料就绪度提示缺少 **劳动关系证明** 类材料。  
3. 推荐路径：整理证据 → 生成文书（仲裁申请书 + 证据清单）。  
4. 办事资源页可跳转 **劳动监察** 属地入口（选江苏省）。
