# Case 002 · 试用期辞退（互联网运营岗）

## 1) 给前台的提示词（描述案件）

我叫周婷，2026年2月入职“启衡数字科技（杭州）有限公司”，岗位是运营专员，合同约定试用期3个月，月薪12000元。  
2026年4月下旬，我在没有收到书面考核结果和正式沟通记录的情况下，被直属主管口头通知“试用期不通过，今天办理离职”。  
公司只愿意按当月出勤结算工资，不愿支付代通知金，也没有给我书面解除证明。  
我手里有录用通知邮件、工资发放记录截图、解除通知截图和一份绩效沟通纪要。  
我想确认：这个解除是否合法，是否可以主张违法解除赔偿或代通知金，需要先仲裁还是先协商。
