# 附录 B：配音制作建议

推荐使用 **阿里 DashScope CosyVoice**（已有 DashScope API Key）：

- **男声**：选择沉稳、有力量的音色
- **参数**：语速 1.0x，音调适中
- **分段录制**：按照六幕分别生成音频文件，便于剪辑对应

备选：Azure TTS（云希/云野），中文自然度高，也支持 SSML 标记控制语速和停顿。

# 附录 C：剪辑要点

| 要点 | 说明 |
|------|------|
| **画面切换** | 旁白说到关键词时切画面，不要提前或延后 |
| **PIP 技术小窗** | 右下角小窗展示架构图/流程图，不要打断主录屏 |
| **鼠标指针** | 录屏时放大鼠标指针，加点击波纹效果（OBS 插件/后期） |
| **关键操作高亮** | 点击按钮时加短暂发光边框（剪映关键帧蒙版） |
| **倒计时/计时器** | 第三幕末尾加入实耗时间数字动画 |
| **BGM 处理** | 第一幕低沉钢琴 → 第二幕科技感电子 → 第三幕保持中性 → 第六幕温暖弦乐渐强 |
